The SWISH chat services allows you to chat with users about a file. SWISH displays an avatar for each user with whom you share an open file. If you select File/Chat ... the current notebook or program gets a chat window displayed below it, preloaded with older chat messages about this file. The Send button menu provides advanced options such as including your query and broadcasting to the shared help chat room (see below).

The shared chat room

The notebook Help.swinb provides a place to find buddies. It can be opened from File/Chat help room ... and messages may be cross-posted to this shared space using Cry for help from the Send button. The message that appears in the shared room contains a link to the file from which it was created.

Status of the SWISH sharing service

What you are seeing are the first steps to allow users of SWISH to communicate. There are many facilities we have in mind, such as:

  • Connect editors for collaborative editing
  • Make teams that connect people also if they are not editing the same file.
  • Allow Prolog programs to exchange messages through the chat room.
